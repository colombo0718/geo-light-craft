# GeoLight Craft｜幾何光構工藝

結合**幾何骨架 × 樹脂薄膜 × LED 發光**的創作型工藝品牌。  
以 low poly 設計語言為核心，發展具藝術性、教育性與商業潛力的產品系列。

---

## 核心技術

| 技術 | 說明 | 特性 |
|------|------|------|
| 鐵絲骨架 + 薄膜成形 | 金屬細線構成封閉幾何框架，低黏度樹脂表面張力成膜 | 輕量、高透光、結構清晰 |
| Low Poly 幾何語言 | 三角面 / 多邊形結構，2D/3D 模型轉實體骨架 | 模組化、可標準化、與數位設計相容 |
| 光學設計（LED） | 微型 LED + 鈕扣電池，透明薄膜導光折射 | 夜間效果強、提升價值感 |

---

## 產品線

| 系列 | 產品形式 | 設計方向 | 定位 |
|------|----------|----------|------|
| 🟢 花卉系列 | 樹脂花吊飾、花卉耳環、幾何花擺飾 | 傳統花型、low poly 花瓣、LED 版本 | 入門主力、建立市場接受度 |
| 🟡 動物系列 | 幾何動物吊飾（貓、狗、狐狸）、耳環 | 幾何輪廓、面與線組合、顏色主題 | 高辨識度、易形成 IP |
| 🔵 幾何結構系列 | 多面體吊飾、幾何水晶、抽象擺飾 | 對稱模組化、部分面成膜留白 | 設計型、禮品市場 |
| 🟣 發光結構系列 | LED 幾何吊飾、發光動物、桌上型發光雕塑 | 光源位置設計、光擴散控制、電池隱藏 | 高附加價值、市場差異化 |

---

## 製作流程

```
設計（low poly 線框）→ 骨架製作（鐵絲彎折）→ 薄膜成形（樹脂沾膜）
→ UV 固化修整 → 組裝（五金 / LED 電池選配）
```
